## 1. WSLの有効化

スタート画面を開き、「Windowsの機能の有効化または無効化」を入力する。

以下のように項目が表示されるので、「Windowsの機能の有効化または無効化」をクリックして開く。

「Linux 用 Windows サブシステム」をチェックして有効化する。

「Hype-V」をチェックして有効化する。
